What is the Quadratic Formula?

Derived by completing the square on ax² + bx + c = 0, the quadratic formula solves for x in any quadratic. The derivation:

  1. ax² + bx + c = 0
  2. x² + (b/a)x = −c/a
  3. Complete the square: x² + (b/a)x + b²/4a² = (b² − 4ac)/4a²
  4. (x + b/2a)² = (b² − 4ac)/4a²
  5. x = (−b ± √(b² − 4ac)) / 2a
