Format

Tag each value with b<base> — e.g. FFb16 is FF in base 16, 1010b2 is 1010 in base 2. Use +, −, *, /. For digit values above 9, use A–Z or parentheses: (10)(11)b20.

Each number is converted to base 10, the arithmetic is performed, and the result is converted to your chosen output base.
